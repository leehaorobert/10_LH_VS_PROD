开始;
block 设定可选股票范围{
取得股票池
(沪深300+创业板指);
剔除当日停牌股票;
取得股票的动态市盈率、净利润增长率、市值;
取得动态市盈率大于0的股票;
取得净利润增长率大于0的股票;
取得市值小于1000亿的股票;
计算股票的PEG值
(动态市盈率/净利润增长率);
按照股票的PEG值进行升序排序;
取得备选股票的行业信息;
限制每个行业的股票数量(3只);
取得PEG排名在11-20的股票;
生成可选股票;
}
block 生成卖出/买入清单{
  取得当前的股票持仓;
  if(当前股票停牌){
    继续持仓该股票(不处理);
  }
  else{
    if(当前股票涨停(>9%)){
      继续持仓该股票(不处理);
    }
    else{
      if(当前股票在可选股票清单){
        继续持仓该股票(不处理);
      }
      else
    }
  }
  生成卖出股票清单;
  生成买入股票清单;
}
block 执行卖出/买入操作{
  执行卖出操作;
  执行买入操作;
}
结束;
